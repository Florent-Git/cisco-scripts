## Usage

* sh
  ```sh
  Usage: main.py [OPTIONS]

  Options:
  -u, --url TEXT        SSH url to connect to the host
  -u, --user TEXT       Cisco SSH user
  -p, ---password TEXT  Cisco SSH password
  --ssh-version [1]
  --help                Show this message and exit.
  ```
